# Signals — Companion Archive

This folder holds the GitHub companion to each *A Somatic Unraveling* essay published on
**[Signals & Seeds](https://signalsandseeds.xyz)**.

Each essay on Signals is human-written organic work. Here, that essay is paired
with its evidence: archival excerpts from the 25-year mothernode, cstreetOS
pattern notes tracing the throughline between past and present writing, and —
where relevant — HealingOS body data from the same window.

The essay is the front door. This is the evidence room.

## How it's organized

One folder per essay:
signals/
[essay-title]/
[essay-title].md — the published essay (or excerpt)
pattern-note.md — cstreetOS analysis: the past-to-present thread
excerpt-[year].md — archival mothernode passage, quoted as-is
healingos-data.md — body data from the same window (when relevant)

## The layers

- **Mothernode** — the writer's own organic work, human-created and human-edited.
  Quoted unedited. Never AI-generated.
- **Analysis (cstreetOS)** — pattern notes generated by reading the mothernode.
  System output, labeled as such. Documents what the writing does; never
  reproduces or extends it.
