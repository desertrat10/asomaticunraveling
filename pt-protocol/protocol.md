# Self-Directed PT Protocol

The self-directed physical-therapy layer of *A Somatic Unraveling*. Three goals,
tracked as a node map. Each box is a node; checking it is the record of landing it.
Care providers are identified by role label only.

**Goals**
- **MT-30** — return to light Muay Thai (Stage 1: technique/movement only) within ~30 days
- **Taper-60** — taper off pain medication within ~60 days, prescriber-directed
- **Recovery Compression** — shorten return-to-baseline time across logged experiments (ongoing)

The body sets the pace. Day targets are planning anchors, not deadlines. Gates open
when the data says they open.

---

## Phase 0 — Assembly (days 0–14)
- [ ] PT3 outreach sent
- [ ] PT3 eval completed (90-min) — provider brief delivered at intake
- [ ] DR1 contacted re: primary doctor lineup
- [ ] Primary doctor confirmed
- [ ] BW1 booked — first 60-min massage + craniosacral
- [ ] Monthly cadence locked: PT3 1–2x / BW1 1x (2–3 professional body sessions/month)

## Phase 1 — Ramp (weeks 1–4)
- [ ] Yoga baseline week logged (known-load baseline capture)
- [ ] Gym re-entry week logged (scapular/core, bands/cables)
- [ ] Baseline stability gate cleared (per training-return protocol)
- [ ] MT Stage 1 entry — technique/movement only, capped session, no bag
- [ ] First clean 48-hr post-session window logged

## Phase 2 — Taper (days 30–60)
- [ ] Taper schedule set with prescriber (RX)
- [ ] Taper start logged as re-baselining event (tagged factor added to daily logs)
- [ ] Day-60 checkpoint: taper complete OR revised with data

## Phase 3 — Experiment engine (ongoing)
- [ ] Recovery-window metric defined (load event → return-to-baseline, in days)
- [ ] Baseline dataset published: networking-event experiments (no-recent-bodywork condition)
- [ ] Bodywork cadence held constant across ≥2 experiment cycles
- [ ] Quarterly recovery-window trend review published

---

## The confound this protocol resolves

The current recovery stall (elevated baseline ~7 days post-event) coincides with the
longest gap in professional bodywork (~30 days since last traditional PT). Recovery
delay and maintenance gap are currently indistinguishable in the data. Locking a monthly
bodywork cadence converts it into a held-constant condition — after which recovery windows
measure the experiments, not the gap.
