# PT Protocol

The self-directed physical-therapy layer of *A Somatic Unraveling*.

This folder holds the live protocol for rebuilding a functional movement baseline with a confirmed multi-level cervical degenerative condition — tracked openly, node by node, as part of the project's patient-sovereignty commitment.

- **[protocol.md](./protocol.md)** — the canonical node map. Three goals, checkboxes marking each landed node. This is the source of truth; the linked GitHub Project board (if present) mirrors it.

Care providers are referred to by role label only (PT3, BW1, DR1, RX). No provider names or clinic links appear anywhere in this repository.

Full project scope, methodology, and provider-facing documentation live in the [wiki](../../wiki).
